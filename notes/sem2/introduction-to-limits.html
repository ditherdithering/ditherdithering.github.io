Definition

A function that approaches a certain value.

Should the graph does not converge on a given \( \lim_{x\to a} \), the answer should be DNE.

Limit Theorems

assume that \( f(x) = L \), and \( g(x) = M \).

Unsorted

the limit of a constant is itself. If \( k \) is a constant then \( \lim_{x \to a}k = k \)

the limit of \( x \) as \( x\to a = a \). \[ \lim_{x\to a}x = a \]

Should a coefficient occur, move it to the first, i.e: \[ \lim_{x\to a}2x^2 = 2\lim_{x\to a}x^2 \]

Constant Multiple Theorem

assuming that if the limits \( f \) and \( g \) exist as \( x\to a \), if multiple functions, then \( \lim_{x \to a} k \cdot f(x) = k \lim_{x\to a} L = k \cdot L \)

Addition Theorem

the sum or difference between two of the functions is the same as adding or subtracting them individually: \[ \lim_{x\to a}(f(x) + g(x)) = \lim_{x\to a}f(x) + \lim_{x\to a}g(x) = L + M \]

Multiplication Theorem

same as addition theorem: \[ \lim_{x\to a}(f(x) \cdot g(x)) = \lim_{x\to a}f(x) \cdot \lim_{x\to a}g(x) = L \cdot M \]

Division Theorem

same as addition theorem: \[ \lim_{x\to a}\frac{f(x)}{g(x)} = \frac{\lim_{x\to a}f(x)}{\lim_{x\to a}g(x)} = \frac{L}{M}, \hspace{1.5em} M \neq 0 \]

Power Theorem

same as division theorem, where \( p \) is any integer: \[ \lim_{x\to a}(f(x))^p = (\lim_{x\to a}f(x))^p= L^p \]

Radical Theorem

same as power theorem, where \( n \) is any integer for root: \[ \lim_{x\to a} \sqrt[n]{f(x)} = \sqrt[n]{\lim_{x\to a}f(x)}= \sqrt[n]{L} \] If the denominator is a radical, multiply both sides by the radical to rationalize

Indeterminate Forms

if \( \lim_{x\to a}f(x) = 0 \) and vice versa, then \( \lim_{x\to a}\frac{f(x)}{g(x)} \) is an indeterminate form of dividing by zero.

However, if there are common factors to both numerator and denominators, then: \[ \lim_{x\to a}\frac{x^2 - 4}{x - 2} \neq 0 \]

Bwaa

The superscript of \( a \) in \( \lim_{x\to a} \) determines where direction is going to, where \( + = \) from the right, \( - = \) from the left.

Factoring Indeterminates

Example 1

\[ \lim_{x\to -3}\frac{x^2 - 7}{x + 3} \]

Example 2

\[ \require{cancel} \lim_{x\to -1}\frac{x^2 + 2x + 1}{x + 1} = \lim_{x\to -1}\frac{\cancel{(x + 1)}(x + 1)}{\cancel{x + 1}} = \lim_{x\to -1}(x + 1) = -1 + 1 = 0 \]

Example 3

\[ \lim_{x\to 1}\frac{x^2 - 1}{\sqrt{x} - 1} \]

LT1 No. 4

\[ \lim_{x\to 5}\frac{x^2 - 2x}{x - 5} = \frac{x(x - 2)}{x - 5} \]

LT1 No. 5

\[ \lim_{r\to -1}\frac{r^3 + 3r^2 + 3r + 1}{r + 1} = \frac{(r + 1)^3}{\cancel{r + 1}} \] \[ (r + 1)^2 = ((-1) + 1)^2 = 0 \]

LT1 No. 6

\[ \lim_{x\to 0}\frac{x^2 - 4x}{x^3 - 3x^2 - 2x} \]

LT1 No. 8

\[ \frac{\sqrt{x + 6} - \sqrt{6}}{x} = \frac{\sqrt{x + 6} - \sqrt{6}}{x} \cdot \frac{\sqrt{x + 6} + \sqrt{6}}{\sqrt{x + 6} + \sqrt{6}} \] \[ \frac{x + 6 + 6}{x \cdot \sqrt{x + 6} + \sqrt{6}} = \frac{12}{\sqrt{6}} \]

LT1 No. 10

\[ \lim_{x\to 6}\frac{x - 6}{x^2 - 12x + 36} = \frac{\cancel{x - 6}}{\cancel{(x - 6)}(x - 6)} \] \[ \frac{1}{x - 6} = \frac{1}{0} \hspace{0.3em} \therefore \lim_{x\to 6} = \text{DNE} \]
