Continuity

Conditions

  1. Must be defined at a point \(f(a)\)
  2. The limit must exist at the point \(\lim_{x\to a^\pm}\)
  3. The value of the function at that point must equal the value of the limit at that point \(f(a) = \lim_{ x \to a^\pm }\)

Discontinuity

  • whole discontinuity
  • jump
  • pull (asymptotes)
  • break

Types

Essential (Nonremovable)

occurs when \(f\) is discontinuous at \(a\) and condition ii is not satisfied.

Removable

occurs when \(f\) satisfies condition ii but fails to satisfy either i or iii.

The discontinuity can be removed by re-defining \(f\).

Continuity at an Interval

Name of Interval Notation Inequality Description Line Representation
Finite and Closed \([a, b]\) \(a \leq x \leq b\) \(\cdot_a \cdot_b\)
Finite and Open \((a, b)\) \(a < x < b\) \(\circ_a \circ_b\)
Finite and Half Open/Closed \((a, b], [a, b)\) \(a < x \leq b, \hspace{.5em} a \leq x < b\) \(\cdot_a \circ_b, \hspace{.5em} \circ_a \cdot_b\)

If \(f\) is continuous on every point on an open interval, we can say \(f\) is continuous on \((a,b)\).

If \(f\) is continuous on every point on a closed interval, we can say \(f\) is continuous on the open interval and \[ \lim_{x\to a^+} f(x) = f(a) \quad \text{and} \quad \lim_{x\to b^-} f(x) = f(b) \]

If \(f\) is continuous on any values of \(x\), i.e. \(\pm\infty\), we can say \(f\) is continuous.
