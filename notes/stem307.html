Conic Section

  • A curve obtained from the intersection of a right circular cone and a plane
  • Contains Parabola, Circle, Ellipse, & Hyperbola

Identifying Conics

\[ Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0 \]

Cases Type of Conic Section
A = C Circle, Point or ∅
A and C have the same sign & A ≠ C Ellipse, Point or ∅
A = 0 or C = 0 Parabola
A and C have opposite signs Hyperbola

∅ = Empty Set

Example

\[ 3x^2 + 5y^2 - 6x + 2y - 9 = 0, \hspace{0.25cm} \therefore \hspace{0.25cm} \text{Ellipse} \]

\[ 3y^2 - 4y = 2x^2 + 9x - 3, \hspace{0.25cm} \therefore \hspace{0.25cm} \text{Hyperbola} \]

\[ 6x^2 + 12x - y + 15 = 0, \hspace{0.25cm} \therefore \hspace{0.25cm} \text{Parabola} \]

Classifying Conic Sections using Discriminant

Cases Types
\(B^2 - 4AC = 0 \) & A/C = 0 Parabola
\(B^2 - 4AC < 0\) & A = C Circle
\(B^2 - 4AC < 0\) & A ≠ C Ellipse
\(B^2 - 4AC > 0\) Hyperbola

Example

\[ y = \frac{1}{x}, \quad -x\left(\frac{1}{x} + y\right) = x(0), \quad -xy + x = 0 \]

Circle & Its Applications

Set of all points P(x, y) in a plane whose distance from a fixed point called the center (h, k) is a constant r called the radius.

Standard Equation

\[ (x - h)^2 + (y - k)^2 = r^2 \]

Example

\[ (x + 2)^2 + (y - 3)^2 = 9 \]

\[ (-1) - h = 2(-1), \quad h = 2 \]

General Equation

\[ Ax^2 + Ay^2 + Cx + Dy + E = 0, \hspace{0.25cm} A \neq 0 \]

\[ x^2 + y^2 + Cx + Dy + E = 0, \hspace{0.25cm} A \neq 0 \]

Parabola

\[ Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0 \]

V(h, k)

\[ (x-h)^2 = 4p(y - k), \hspace{0.2em} \text{where:} \]

p > 0 → Upward
p < 0 → Downward
D: y = k - p

\[ (y - k)^2 = 4p(x - h), \hspace{0.2em} \text{where:} \]

p > 0 → Right
p < 0 → Left
D: x = h - p

\[ x^2 = 4py, \hspace{0.2em} \text{where:} \]

p > 0 → Upward
p < 0 → Downward
D: y = -p

\[ y^2 = 4px, \hspace{0.2em} \text{where:} \]

p > 0 → Right
p < 0 → Left
D: x = -p

Ellipse and its Applications

A set of all points in a 2D plane where the sum of the distances from two distinct fixed points \( F_1 \) and \( F_2 \) are constants.

\[ |Q_{(x_1, y_1)}F_1| + |Q_{(x_1, y_1)}F_2| = |R_{(x_2, y_2)}F_1| + |R_{(x_2, y_2)}F_2| \]

Properties

  • Center
  • Vertices → a → \( (h, k \pm a) \)
  • Co-Vertices → b → \( (h \pm b, k) \)
  • Foci → c → \( (h, k \pm c) \)
  • Length of the Major Axis → 2a
  • Length of the Minor Axis → 2b
  • Principal Axis

Equations

Standard, C(0,0):

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \hspace{0.2em} \text{(horizontal)} \hspace{0.2em} \text{or} \hspace{0.2em} \frac{x^2}{b^2} + \frac{y^2}{a^2} = 1 \]

Note: \( b^2 + c^2 = a^2 \), \( a > b \)

Standard, C(h,k):

\[ \frac{(x - h)^2}{a^2} + \frac{(y - k)^2}{b^2} = 1 \hspace{0.2em} \text{or} \hspace{0.2em} \frac{(x - h)^2}{b^2} + \frac{(y - k)^2}{a^2} = 1 \]

Example

\[ F_1 = (0, 3), \hspace{0.2em} F_2 = (0, -3), \hspace{0.2em} \text{axis} = 8 \]

\[ b = \frac{\text{axis}}{2} = \frac{8}{2} = 4 \]

\[ c = \sqrt{(0 + 0)^2 + (-3-3)^2} = \sqrt{36} = \frac{6}{2} = 3 \]

\[ a^2 = b^2 + c^2 = \sqrt{4^2 + 3^2} = \sqrt{25} = 5 \]

\[ \frac{x^2}{b^2} + \frac{y^2}{a^2} = 1 \]

\[ \frac{x^2}{16} + \frac{y^2}{25} = 1 \]

\[ F_1 = (0, 0), \hspace{0.2em} F_2 = (8, 0), \hspace{0.2em} \text{axis} = 16 \]

\[ c = |\vec{F_1F_2}| = 8 \]

\[ \frac{x^2}{1^2} + \frac{y^2}{3^2} = 1, \hspace{0.2em} C(1, -3) \]

General Equation of an Ellipse

\[ Ax^2 + Cy^2 + Dx + Ey + F = 0, \hspace{0.2em} \text{no arranging} \]

Where \( A \) and \( C \) > 0

Example

\[ 25x^2 + 16y^2 + 150x - 128y - 1119 = 0 \]

\[ (25x^2 + 150x) + (16y^2 - 128y) = 1119 \]

\[ 25(x^2 + 6x + ?) + 16(y^2 - 8y + ?) = 1119 \]

\[ 25(x^2 + 6x + 9) + 16(y^2 - 8y + 16) = 1119 + 9(25) + 16^2 \]

\[ 25(x + 3)^2 + 16(y - 4)^2 = 1600 \]

\[ \frac{25(x + 3)^2 + 16(y - 4)^2}{1600} = \frac{1600}{1600} \]

\[ \frac{(x + 3)^2}{64} + \frac{(y - 4)^2}{100} = 1 \]

Hyperbolas and its Applications

A set of all points in a plane such that the difference of whose foci are constant.

Properties

  • Center
  • Vertices: intersection of the principal axis of the hyperbola
  • Co-Vertices: endpoints of the conjugate axis
  • Foci: located inside the curves, constant
  • Principal axis: line passing through foci
  • Transverse Axis: line connecting vertices
  • Conjugate Axis: line connecting co-vertices
  • Asymptote: guides for graphing the hyperbola, runs through the middle
    • \( y = \pm \frac{b}{a}(x - h) + k \hspace{0.2em} \text{or} \hspace{0.2em} \frac{a}{b} \)
  • Auxiliary Rectangle: line of the asymptote

Standard Form

C(0,0)

\[ \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 \]

Where:

  • Transverse axis = 2a
  • Vertices coordinates: \( (\pm a, 0) \)
  • Conjugate axis = 2b
  • Co-vertices coordinates: \( (\pm b, 0) \)
  • Distance between foci = 2c, where \( c^2 = a^2 + b^2 \)
  • Foci coordinates: \( (0, \pm c) \)
  • Asymptote equation: \( y = \pm \frac{a}{b}x \hspace{0.2em} \text{v.v} \)
C(h,k)

\[ \frac{(x - h)^2}{a^2} - \frac{(y - k)^2}{b^2} = 1 \]

Where:

  • Transverse axis = 2a
  • Vertices coordinates: \( (h \pm a, k) \)
  • Conjugate axis = 2b
  • Co-vertices coordinates: \( (h \pm b, k) \)
  • Distance between foci = 2c, where \( c^2 = a^2 + b^2 \)
  • Foci coordinates: \( (h, k \pm c) \)
  • Asymptote equation: \( y = \pm \frac{a}{b}(x - h) + k \hspace{0.2em} \text{v.v} \)
